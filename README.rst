This is a theme for pelican_ static site generator


Roadmap
=======

v1.0
----
- 10/10 `Opquast Reporting`_
- 404.html
- 100% `page speed test`_ optimisation
- `w3c html5 valid`_
- `w3c css3 valid`_
- `w3c unicorn valid`_
- embed all the ressources ( avoid cdn and google )
- limit the ressources weight ( optimised )
